# Запуск бота (Windows)

## 1) Подготовка окружения

1. Установите Python 3.11+.
2. В корне проекта создайте файл `.env` (на основе `.env.example`).
3. Заполните обязательные значения:
   - `BOT_TOKEN`
   - `BOT_USERNAME`
   - `ADMIN_CHANNEL_ID`
   - `FERNET_KEY`
   - `APP_SECRET`

## 2) Установка зависимостей

```
pip install -r requirements.txt
```

## 3) Инициализация базы

База создается автоматически при первом запуске.

## 4) Запуск бота

```
python -m bot.main
```

## 5) Проверка

- В Telegram откройте бота и отправьте `/start`.
- Пройдите paywall и оформите Stars-оплату.
- После оплаты получите инвайт в канал.

## 6) Частые проблемы

- Если бот не получает события входа/выхода, убедитесь, что он админ канала.
- Если не работает Stars-оплата, убедитесь, что используете `currency="XTR"` и `provider_token=""`.
